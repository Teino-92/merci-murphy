# Scope

## In Scope (MVP — V1)

- [ ] Page Accueil (hero, services, valeurs, témoignages, infos pratiques)
- [ ] Page Concept / À propos (histoire, équipe, valeurs)
- [ ] Pages Services × 5 : Toilettage, Bains, Crèche, Éducation, Ostéo
- [ ] Page Réservation (formulaire prise d'infos + embed Calendly)
- [ ] Page Contact (adresse, horaires, formulaire, Google Maps)
- [ ] Section E-shop (catalogue Shopify via Storefront API)
- [ ] Capture leads + newsletter (Supabase)
- [ ] Email de confirmation automatique (Resend)
- [ ] CMS Sanity (tarifs, FAQ, équipe, témoignages éditables sans dev)
- [ ] SEO on-page (metadata, sitemap, structured data LocalBusiness)
- [ ] Analytics RGPD (Plausible)
- [ ] Design responsive mobile-first

## Out of Scope (V2)

- Paiement en ligne avec empreinte bancaire
- Système de réservation avec gestion de créneaux en temps réel
- Espace client / compte utilisateur
- Blog / journal de marque
- Intégration Ankorstore (sync stock grossiste)
- Programme de fidélité
- Nouvelles adresses / franchises

## Non-Goals

- On ne recrée PAS Shopify — le checkout reste sur Shopify natif
- On n'intègre PAS un back-office custom — Sanity est le seul CMS
- On ne fait PAS d'app mobile
- On ne gère PAS les no-shows directement (pas de validation en ligne V1)
