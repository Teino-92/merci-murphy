# Mode d'emploi — Gestion du contenu Merci Murphy

Ce guide explique comment modifier le contenu du site merci murphy® sans toucher au code.
Tout se passe sur **Sanity Studio** — c'est votre outil d'édition, comme un Google Docs mais pour le site.

---

## 1. Comment accéder à Sanity Studio

Ouvrez votre navigateur (Chrome, Safari, peu importe) et allez sur :

```
https://mercimurphy.com/studio
```

Connectez-vous avec votre compte Sanity (email + mot de passe).

> **Si vous n'avez pas de compte**, demandez à votre développeur de vous en créer un.

---

## 2. L'interface — à quoi ça ressemble

Une fois connecté, vous voyez une barre à gauche avec plusieurs sections.
Chaque section correspond à une partie du site.

| Ce que vous voyez      | Ce que ça contrôle                                     |
| ---------------------- | ------------------------------------------------------ |
| **Service**            | Les pages de vos services (toilettage, bains, crèche…) |
| **Membre de l'équipe** | Les fiches de l'équipe                                 |
| **Témoignage**         | Les avis clients affichés sur le site                  |
| **Paramètres du site** | Adresse, téléphone, horaires, liens                    |

---

## 3. Modifier un service

Les services sont les pages comme "Toilettage", "Les Bains", "La Crèche", etc.

**Pour modifier un service :**

1. Cliquez sur **Service** dans la barre de gauche
2. Vous voyez la liste de vos services — cliquez sur celui que vous voulez modifier
3. Modifiez ce dont vous avez besoin (voir le détail ci-dessous)
4. Cliquez sur le bouton **Publier** en bas à droite — **sans cliquer sur Publier, rien ne change sur le site**

### Ce que vous pouvez modifier dans un service

- **Titre** — Le nom du service (ex : "Toilettage")
- **Accroche courte** — La petite phrase qui résume le service (maximum 160 caractères)
- **Image principale** — La photo qui illustre le service
- **Notre approche** — Le texte de présentation, comme un éditeur de texte classique (gras, italique, listes…)
- **Déroulé du rendez-vous** — Optionnel, pour expliquer le déroulement
- **Tarifs** — La liste des prestations avec leur prix (voir ci-dessous)
- **FAQ** — Les questions/réponses fréquentes
- **Lien Calendly** — L'URL de réservation pour ce service

### Ajouter ou modifier un tarif

1. Dans la section **Tarifs**, cliquez sur **Ajouter**
2. Remplissez :
   - **Prestation** — Le nom (ex : "Petit chien jusqu'à 5kg")
   - **Prix** — Le tarif (ex : "45 €")
   - **Précision** — Optionnel, une note en plus petit (ex : "selon état du poil")
3. Cliquez **Publier**

Pour **supprimer** un tarif, cliquez sur les trois petits points à droite de la ligne et choisissez "Supprimer".

Pour **réordonner** les tarifs, glissez-déposez les lignes avec le symbole ≡ à gauche.

---

## 4. Gérer l'équipe

1. Cliquez sur **Membre de l'équipe** dans la barre de gauche
2. Cliquez sur un membre existant pour le modifier, ou sur **Créer** pour en ajouter un
3. Remplissez :
   - **Nom** — Prénom et nom
   - **Rôle** — Ex : "Toiletteuse", "Responsable crèche"
   - **Bio** — Courte présentation (optionnel)
   - **Photo** — Importez une photo depuis votre ordinateur
   - **Ordre d'affichage** — Un chiffre pour décider dans quel ordre ils apparaissent (1 = en premier)
4. Cliquez **Publier**

---

## 5. Ajouter un témoignage client

1. Cliquez sur **Témoignage** dans la barre de gauche
2. Cliquez sur **Créer**
3. Remplissez :
   - **Auteur** — Prénom du client (ex : "Sophie M.")
   - **Note** — Entre 1 et 5
   - **Témoignage** — Le texte de l'avis
   - **Service concerné** — Choisissez le service dans la liste
   - **Date** — La date de l'avis
4. Cliquez **Publier**

---

## 6. Modifier les informations générales du site

Ici vous gérez l'adresse, le téléphone, les horaires et les liens.

1. Cliquez sur **Paramètres du site** dans la barre de gauche
2. Vous voyez un seul document — cliquez dessus
3. Modifiez ce que vous voulez :

### Adresse et contact

- **Adresse** — Le numéro et la rue (ex : "12 rue du Bac")
- **Ville** — Ex : "Paris"
- **Code postal** — Ex : "75007"
- **Téléphone** — Le numéro de contact
- **Email** — L'email de contact

### Horaires d'ouverture

Les horaires sont organisés par **sections** (ex : "Boutique et les bains", "La crèche").

Pour modifier une ligne horaire :

1. Cliquez sur la section concernée (ex : "Boutique et les bains")
2. Cliquez sur la ligne à modifier (ex : "Mardi – Vendredi")
3. Changez les **Jours** et les **Heures**
4. Cliquez **Publier**

Pour ajouter une nouvelle section d'horaires : cliquez **Ajouter un élément** en bas de la liste des sections.

### Liens

- **Instagram** — L'URL de votre page Instagram
- **Google Maps** — Le lien vers votre fiche Google Maps
- **Calendly** — Le lien de réservation général

4. Cliquez **Publier** quand vous avez terminé

---

## 7. Importer une image

Quand vous voyez un champ **Image**, voici comment faire :

1. Cliquez sur **Importer une image**
2. Choisissez "Télécharger" et sélectionnez le fichier sur votre ordinateur
3. Une fois l'image chargée, vous pouvez **déplacer le point de cadrage** (le petit cercle) pour choisir quelle partie de l'image sera visible quand elle est rognée
4. Cliquez **Valider**

> **Conseil** : Utilisez des photos en format JPG ou WebP, pas trop lourdes (moins de 2 Mo idéalement).

---

## 8. Règle d'or — toujours cliquer sur Publier

Toutes vos modifications sont **sauvegardées automatiquement** pendant que vous travaillez — vous ne perdrez rien si vous fermez la page accidentellement.

Mais elles ne sont **visibles sur le site** que quand vous cliquez sur **Publier**.

Vous verrez un bouton vert **Publier** en bas à droite de l'écran. Tant qu'il est là, vos modifications ne sont pas encore en ligne.

---

## 9. En cas de problème

- **J'ai fait une erreur** — Pas de panique. Cliquez sur l'icône d'historique (horloge) en haut à droite pour revenir à une version précédente.
- **Je ne vois pas mes modifications sur le site** — Vérifiez que vous avez bien cliqué sur **Publier**. Si oui, attendez 1-2 minutes et rechargez la page du site.
- **Je ne peux pas me connecter** — Contactez votre matteo.garbugli@yahoo.it pour réinitialiser votre mot de passe.
